# Global Settings Endpoints

Here's a recap of the global settings endpoints:

### Public Endpoints

*   `GET /api/global-settings/public`: Get public global settings.

### Protected Endpoints (Basic Auth Required)

*   `GET /api/global-settings/`: Get all global settings.
*   `GET /api/global-settings/:key`: Get a specific global setting by key.
*   `PUT /api/global-settings/:key`: Update a specific global setting by key.
*   `POST /api/global-settings/`: Create a new global setting.
*   `DELETE /api/global-settings/:key`: Delete a specific global setting by key.
